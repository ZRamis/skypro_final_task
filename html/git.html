Git

  • ls – вывести список файлов и папок в текущей директории
  • cd – переместиться между папками
  • pwd – вывести текущую рабочую директорию
  • mkdir – создать папку
  • mv – переместить или переименовать файл
  • cls – очистить терминал от лишнего кода
  • git init – создание нового репозитория Git в пустой директории или превращение существующей директории в новый репозиторий Git
  • git push – выгрузка сделанных изменений в удаленный репозиторий
  • git pull – получение изменений из удаленного репозитория на локальную машину
  • git status – просмотр списка измененный файлов
  • git commit – сохранение изменений в Git
  • git log – просмотр истории коммитов в репозитории
  • git branch -a – показать все локальные и удаленные ветки
  • git branch -r – показать только удаленные ветки
  • git branch – показать только локальные ветки
  • git fetch – получение последних изменений из удаленного репозитория
